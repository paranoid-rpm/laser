## LaserTech — Лазерные технологии и их использование

Многостраничный информационный сайт о лазерных технологиях.

### Структура проекта

```
├── index.html              # Главная страница
├── pages/
│   ├── theory.html         # Теория лазерного излучения (встроенные интерактивы)
│   ├── types.html          # Типы лазеров (встроенный симулятор луча)
│   ├── applications.html   # Применение лазеров
│   ├── history.html        # История развития
│   ├── safety.html         # Безопасность (интерактив: поражение глаза)
│   └── interactive.html    # Полная интерактивная лаборатория + SVG-схемы
├── css/
│   ├── style.css           # Основные стили
│   └── animations.css      # Анимации и эффекты
├── js/
│   ├── main.js             # Интерактивность сайта
│   └── animations.js       # Интерактивные симуляции (canvas)
└── README.md
```

### Особенности

- Тёмная тема с неоновыми акцентами (cyan, purple, pink)
- Glassmorphism-эффекты (размытие, прозрачные карточки)
- Анимированный фон (сетка, лазерные лучи, частицы)
- Glitch-эффект на заголовках
- Анимация появления при скролле (Intersection Observer)
- Подсветка карточек при наведении мыши
- Интерактивные симуляции:
  - Симулятор луча (управление мышью)
  - Вынужденное излучение (клик → фотон)
  - Оптический резонатор (клик → волновой пакет)
  - Энергетические уровни (клик → накачка)
  - Безопасность: поражение глаза (фокусировка + импульс)
- Адаптивный дизайн (мобильные устройства)

### Запуск

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
npx serve .
```
